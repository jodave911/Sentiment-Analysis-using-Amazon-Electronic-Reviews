# Sentiment Analysis on Electronics Reviews with Visualization

## Overview
This project aims to classify Amazon electronics reviews as either **positive** or **negative** using **sentiment analysis** techniques. The project involves **data preprocessing**, **model training**, and **visualization** to interpret the model's performance. This tool can help businesses understand consumer opinions at scale.
